# STL_Template
程式設計期末自主學習

編譯執行流程:
持續輸入整數1~5可執行對應功能
輸入0或者錯誤格式(系統判定為0)會使程式結束
輸入超過5的整數為非法輸入，需重新輸入

1:新增學生資料
    輸入學號，如果學號重複，代表該學生已存在，將拒絕存取並輸出錯誤訊息
    接著輸入名字與成績，若成績超過滿分(100)為錯誤，需重新輸入

2~5:需有學生資料才能執行
2:輸出資料
    以表格狀列出學生資料，map會預設以學號字串排序列出
3:成績排序
    將資料加入vector裡並用sort排序，最後輸出排序後結果
    由排序的function與輸出的function各司其職來執行
4:學號查詢
    輸入學號，如果有該學號則輸出對應學生資料
    反之提示未存取
5:輸出統計資料
    計算平均(輸出時由1.0*總和/人數)、最高分、最低分、及格人數與不及格人數
